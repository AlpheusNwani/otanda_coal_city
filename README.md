# OTANDA COAL CITY LIMITED - Corporate Website

# Project Overview
This is a responsive multi-page corporate website built for **OTANDA COAL CITY LIMITED**, a real estate and construction company specializing in:

- Land sales
- Property development
- Property management
- Construction services
- Equipment rental

The website was designed using modern front-end technologies to ensure responsiveness, usability, and a professional business appearance.


# Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript
